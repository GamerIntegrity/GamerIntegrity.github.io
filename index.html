Download and use

Start here: this is the section for players, admins, and community staff who just want to run a PC check and open the report.
Step 1
Download the release

Grab the latest Gamer Integrity release from the project page.

Open GitHub project
Step 2
Extract the ZIP

Put the folder somewhere simple, such as Desktop or Downloads. Keep the files together.

Step 3
Run as administrator

Right-click GamerIntegrity.exe and choose Run as administrator. Some checks need elevated access.

Step 4
Open the report

Use Open Report for the full report or Open Redacted Report when privacy matters.

Unsigned build warning: if the release is unsigned, Windows SmartScreen or antivirus prompts may appear. Download only from the official project page.

What it checks

Source / builds
Cheat source and build artifacts

Looks for names and project structures tied to cheat source code, build outputs, mappers, injectors, spoofers, and cleaners.

Execution
Execution traces

Checks Windows execution artifacts such as Prefetch and AmCache where available.

Browser
Downloads and history

Looks for cheat-related downloads, source URLs, provider domains, and relevant search/history snippets.

Installed tools
Tooling context

Flags configured cheat tools and weak dual-use tools, with lower weighting for tools that are not cheating evidence by themselves.

System
Review context

Captures driver inventory, boot-security posture, hardware identity records, and USB history.

Sharing
Redacted output

Creates a privacy-friendly report that masks usernames, computer names, serials, MACs, USB IDs, and report paths.

Report guide

The HTML report is the easiest file to review. The JSON file is for machine-readable evidence and deeper review.

SectionWhat it meansWhat to look at
DeterminationThe overall finding level.Score, confidence, and basis text.
Case SummaryThe short reviewer summary.Corroborated chains and review priority.
Evidence timelineDated events grouped by day.Downloads, execution traces, and timing.
ScoringHow evidence contributed to the result.Top factors and category breakdown.
Detailed findingsAll findings by category.High and Critical items first.
Integrity manifestSHA-256 hashes for report files.Use it to verify reports were not changed.

Reviewer guide

Be fair: do not judge a player from one weak clue. Strong cases usually have multiple evidence types pointing in the same direction.
01
Start with Case Summary

It tells you what matters before the long evidence tables.

02
Check the timeline

Look for downloads followed by extracted projects or injector execution.

03
Review direct artifacts

Source files, build folders, mapper files, and injector traces are higher-signal.

04
Keep context

Dual-use tools alone are not the same as cheat evidence.

Known limitations

Human review required: Gamer Integrity is a review tool, not a magic verdict machine. Use it with policy, context, and human review.
  • Names-only file scanning does not open or inspect file contents.
  • AmCache may exist but be unreadable from a live OS.
  • Browser-history timestamps may be missing for some records.
  • Hardware spoofing cannot be proven from one scan without a trusted prior baseline.
  • Dual-use tools like debuggers or disassemblers can be legitimate and should not be treated as cheating by themselves.

FAQ

Does Gamer Integrity upload my files?

No. The scan is local and writes reports on the PC.

Why does it need administrator access?

Some Windows evidence areas need elevated access to read properly.

Does a Critical report automatically prove cheating?

It means the scan found strong local evidence. Staff should still review the details and follow their own process.

Why do I see a SmartScreen warning?

If the build is unsigned, Windows may warn on first run. Download only from the official project page.
